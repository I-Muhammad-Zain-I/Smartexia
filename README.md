# Smartexia

Smartexia is an innovative e-commerce mobile application designed for buying and selling smart appliances. It provides a seamless platform for users to manage, buy, and sell a wide range of smart devices that enhance home automation and connectivity. Smartexia features a user-friendly interface and robust functionalities to cater to all your smart appliance needs.

## Features
- **Product Catalogue**: Browse a comprehensive catalogue of smart appliances.
- **User Profile**: Create and manage personal profiles with purchase history.
- **Add to Cart**: Add and manage items in your shopping cart.
- **Payment Gateway**: Secure payments through Stripe integration.
- **Advanced Filters**: Search and filter products by category, brand, price, and compatibility.
- **Compatibility Checker**: Verify the compatibility of smart appliances with other devices.
- **Admin Panel**: Manage products, orders, and users through an intuitive admin interface.
- **Notifications**: Receive notifications for order status and special offers.

## Technologies Used
- **Frontend**: React Native, Redux, HTML5, CSS3, JavaScript
- **Backend**: ASP.NET Core WEB API, Entity Framework Core, C#
- **Database**: Supabase (postgresql)
- **Payment Gateway**: Stripe
- **Authentication**: JWT, OAuth

## Installation
1. **Clone the Repository**
    ```bash
    git clone https://github.com/yourusername/Smartexia.git
    cd smartexia
    ```

## Usage
1. **Sign Up / Sign In**: Create an account or log in to an existing account.
2. **Browse Products**: Explore the product catalogue and use filters to find desired smart appliances.
3. **Add to Cart**: Add products to your cart and proceed to checkout.
4. **Payment**: Complete your purchase securely using Stripe.
5. **Compatibility Checker**: Use the compatibility checker feature to ensure your smart devices work together seamlessly.

## Compatibility Checker
The Compatibility Checker feature allows users to verify the interoperability of smart appliances with other devices in their home setup. It supports various communication protocols like Zigbee, Z-Wave, Wi-Fi, and Bluetooth, ensuring a cohesive smart home ecosystem.
